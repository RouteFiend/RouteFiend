  80.  * java.lang.RuntimeException or java.lang.Error after the last
  81.  * java_last_exception_clear() invocation or, if no

  90.  * caught on PHP level, and

  92.  * error at the end of the php script (unless the error condition is
  93.  * cleared using java_last_exception_clear()).

  96.  * java.lang.RuntimeException is an exception caused by a bug in your
  97.  * program, for example java.lang.NullPointerException.
  98.  * java.lang.Error marks a serious problem, for example
  99.  * java.lang.OutOfMemoryError.

  447. /**

  451.  * set with engine.set("key", val) can be fetched from PHP with
  452.  * java_context()->get("key"). Values set with
  453.  * java_context()->put("key", java_closure($val)) can be fetched from
  454.  * Java with engine.get("key"). The get/put methods are convenience shortcuts for getAttribute/setAttribute. Example:
  455.  * <code>

  460.  *
